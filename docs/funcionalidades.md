# Funcionalidades do Sistema

## Funcionalidades Planejadas

O sistema contará com as seguintes funcionalidades:

### Visualização de vagas
O usuário poderá visualizar as vagas disponíveis no estacionamento.

### Reserva de vaga
O usuário poderá selecionar uma vaga disponível e realizar uma reserva.

### Controle de ocupação
Após a reserva, a vaga ficará marcada como ocupada no sistema.

### Cancelamento de reserva
O usuário poderá cancelar uma reserva previamente realizada.

### Painel de gerenciamento
Interface para visualização geral das vagas e controle de reservas.
