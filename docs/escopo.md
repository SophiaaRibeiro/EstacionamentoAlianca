# Escopo do Projeto

## Descrição

O Sistema de Reserva de Estacionamento é uma aplicação web em desenvolvimento que tem como objetivo facilitar o gerenciamento de vagas em um estacionamento.

A proposta do sistema é permitir a visualização das vagas disponíveis e a realização de reservas de forma organizada, evitando conflitos de ocupação e melhorando o controle do fluxo de veículos.

O projeto está sendo desenvolvido como parte de estudo em desenvolvimento web, com foco na organização do projeto, documentação de funcionalidades e estruturação do sistema.

---

## Problema que o sistema resolve

Em estacionamentos sem controle organizado, podem ocorrer problemas como:

- Conflitos na utilização das vagas
- Falta de controle sobre vagas disponíveis
- Dificuldade para gerenciar reservas

O sistema busca resolver essas questões por meio de uma interface simples que permita visualizar e reservar vagas de forma organizada.

---

## Objetivos do projeto

- Permitir a visualização das vagas disponíveis
- Organizar o processo de reserva de vagas
- Facilitar o gerenciamento do estacionamento
- Melhorar o controle de ocupação das vagas
