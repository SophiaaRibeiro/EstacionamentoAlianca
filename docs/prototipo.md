# Prototipagem do Sistema

A estrutura inicial das telas do sistema foi planejada com foco em simplicidade e facilidade de uso.

## Telas planejadas

- Página inicial
- Tela de como funciona o fluxo de reserva
- Tela de agendamento de vaga
- Tela de avaliações
- Tela de Contato
