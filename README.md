# Sistema de Reserva de Estacionamento

🚧 Projeto em desenvolvimento

## Descrição do Projeto

Este projeto tem como objetivo o desenvolvimento de um sistema web para gerenciamento de reservas de vagas em estacionamento. A proposta é permitir a organização das vagas disponíveis, facilitar o controle de reservas e melhorar o gerenciamento do fluxo de veículos.

O sistema está sendo desenvolvido como projeto de estudo, com foco na prática de desenvolvimento web, organização de projetos e documentação de funcionalidades.

## Objetivo

Criar uma aplicação web que permita:

- Organização e controle de reservas de vagas
- Visualização da disponibilidade do estacionamento
- Gerenciamento das vagas e horários de utilização
- Melhor controle do fluxo de entrada e saída de veículos

## Funcionalidades Planejadas

- Cadastro de reservas de vagas
- Visualização de vagas disponíveis
- Controle de horários de entrada e saída
- Painel para gerenciamento de vagas
- Interface intuitiva para usuários

## Tecnologias Utilizadas
